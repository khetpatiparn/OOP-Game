### Puzzle, Maze Game

Control a square object, avoid obstacles, and pass through various levels.

## Details

### How to Play

1. Drag the mouse to move the square character.
2. Dodge obstacles
3. Reach the finish line.

### Features

* Many levels
* Challenging obstacles
* Simple graphics
* Easy to play

### Tools Used

* Java AWT & Swing
